Scene File Reference

About

A reference to how scene nodes are serialized in json. The editor configures these files for the engine to read.

Node Types

    "name": "Node Example",
    "type": "Node",
    "tags": [],
    "external_scene_source": "",
    "components": [],
    "children": []
}

Misc

Scriptable

A scriptable component can be attached to any NodeType.

{
    "name": "Scriptable Example",
    "type": "Node",
    "tags": [],
    "external_scene_source": "",
    "components": [
      {
        "scriptable_class": {
          "class_path": "src.main",
          "class_name": "Main"
        }
      }
    ],
    "children": []
}

Children

All nodes can have nested children.

{
    "name": "Parent Example",
    "type": "Node",
    "tags": [],
    "external_scene_source": "",
    "components": [],
    "children": [
      {
        "name": "Child Example",
        "type": "Node",
        "tags": [],
        "external_scene_source": "",
        "components": [],
        "children": []
      }
    ]
}
