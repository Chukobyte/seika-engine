# Fighting Game Proto

A small project to prototype a fighting game and online play.

### Controls

#### Player One

* Move Left: Left Key
* Move Right: Right Key
* Attack: Space

#### Player Two

* Move Left: A
* Move Right: D

#### General

* Shake camera: G
* Camera zoom in: O
* Camera zoom out: P
* Camera move left: J
* Camera move right: L
* Camera move up: I
* Camera move down: K

* Go back to title screen: ESC
* Exit game from title screen: ESC
